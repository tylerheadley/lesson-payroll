Swim Lesson Payroll Converter

This site can be used to easily convert a file exported from CivicRec to payroll for the SOCP Swim Lesson Program.

  1. In CivicRec, go to "Reports" > "Reservation Reports" > "Facility Reservations by Factility".
  2. Set the date range in accordance with the pay period.
  3. Under "Filters", check "1:1 Swim Coaching", "Swim Instructors", and "Customer Reservation".
  4. Under "Fields", check "Location", "Facility", "Start Date", "Start Time", "End Time", and "Full Name".
  5. Export the report as a normal spreadsheet.
